# Fiets Inruil Calculator – Website (Productie-klaar)

## Publicatie
- Upload deze map als repo naar GitHub en zet **Settings → Pages** aan (branch: `main`, root).

## Video op de homepage
- **YouTube**: vervang de `iframe src` in `index.html` bij sectie `#video`.
- **Lokaal MP4**: plaats jullie video in `/videos/intro.mp4` en zet de `<video>`-variant aan (iframe uitcommentariëren).

## Aanvraagknop
De CTA "naar calculator/aanvragen" linkt naar `https://fietsinruil.fietsserviceid.nl/`.
